1 Introduction

This project was written on weiyis-MacBook-Pro.local with Statamarkdown version 0.6.0, knitr version 1.31 and Stata SE version 15. The package Statamarkdown is used to produce documents based on Stata using R Markdown.

Statamarkdown can be installed from github.com.

library(devtools) # before this you may need to install devtools
install_github("hemken/Statamarkdown")

More tutorials for Statamarkdown can be found in Using Statamarkdown.
